A short description of Javascript

In his section you can read a short description of the programming language.

According to Wikipedia JavaScript ,often abbreviated as JS, is a programming language that conforms to the ECMAScript specification.JavaScript is high-level, often just-in-time compiled, and multi-paradigm. It has curly-bracket syntax, dynamic typing, prototype-based object-orientation, and first-class functions.

Alongside HTML and CSS, JavaScript is one of the core technologies of the World Wide Web. JavaScript enables interactive web pages and is an essential part of web applications. The vast majority of websites use it for client-side page behavior, and all major web browsers have a dedicated JavaScript engine to execute it.

As a multi-paradigm language, JavaScript supports event-driven, functional, and imperative programming styles. It has application programming interfaces (APIs) for working with:

  • text
  • dates
  • regular expressions
  • standard data structures
  • and the Document Object Model (DOM)

However, the language itself does not include any input/output (I/O), such as networking, storage, or graphics facilities, as the host environment (usually a web browser) provides those APIs.

JavaScript engines were originally used only in web browsers, but they are now embedded in some servers, usually via Node.js. They are also embedded in a variety of applications created with frameworks such as Electron and Cordova.

Variables in Javascript

In this section, you can read a short description of Javascript variables.

Variables in JavaScript can be defined using either the var, let or const keywords. JavaScript variables are containers for storing data values.

The basic variable type is var.

In the following example exampleVariable contains the sentence: Hi, I'm an example variable!

The value of it can be changed, so the second alert window will display the new value, which can even be of a different type

        
  var exampleVariable = "Hi, I'm an example variable!";
  alert(exampleVariable);
  exampleVariable = 4;
  alert(exampleVariable);

A second type is const. It can't be reassigned, so as the following code runs, it will give an error message, when reaches line 3:


  const exampleVariable = "Hi, I'm an example variable!";
  alert(exampleVariable);
  exampleVariable = "I will give you an error message!";
  alert(exampleVariable);

The variable type let can only be used in scope, for example:


  function myFunction() {
    let exampleVariable = "I am used in the scope of myFunction()!";
    // code here CAN use carName, so:
    alert(exampleVariable);
  }
  //...this will run:
  myFunction();
  //...but this will give you an error message:
  alert(exampleVariable);

Javascript data types

In this section, you can learn about Javascript data types

JavaScript variables can hold several data types: numbers, strings, objects (such as arrays) and other things, for example:


  var exampleNumber = 99;                                     // Number
  var exampleString = "I am a string!";                       // String
  var exampleArray = [0,1,2,3];                               //Array
  var exampleObject = {firstName:"John", lastName:"Doe"};     // Object

Strings

In this section, you will read about strings and some basic examples of code.

Strings can be used to store and manipulate text, for example:


  var exampleString1 = "I am a string!";
  var exampleString2 = " So am I!";
  var exampleString3 = " And we can do so many things together!";
  alert(exampleString1 + exampleString2 + exampleString3);

  var exampleArray = ["I can contain", " text and other stuff", 3, true];

The typeof operator

Here, you can read about the typeof operator and why it is important to know about it.

The typeof operator returns the type of a variable, object, function or expression. This can be important, because in Javascript, a string can become a number or boolean in a blink of an eye


  var iCanChange = "Now I am a string!";
  alert(typeof iCanChange);
  //Now, it will become a number:
  iCanChange = 99;
  alert(typeof iCanChange);
  //Now, an array, which is an object:
  iCanChange = ["I am an array", " which is an object"];
  alert(typeof iCanChange);
